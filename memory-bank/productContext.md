# Product Context

## Why This Exists
This project is a competition entry showcasing creative work on a 19×19 RGB LED grid through a retro game console interface.

## Problems It Solves
- Demonstrates LED matrix programming and visualization
- Shows interactive game development on constrained hardware
- Provides an engaging, nostalgic gaming experience
- Showcases both technical skill and creative design

## How It Works

### User Experience Flow
1. **Boot** → Colorful LED animation (3 seconds)
2. **Menu** → Carousel with 4 games, navigate with arrow keys
3. **Game** → Play selected game, ESC returns to menu
4. **Customization** → Adjust LED appearance in real-time

### Key Features
- **Visual Styles**: Toggle between realistic circular LEDs with glow effects and retro square pixels
- **Runtime Adjustable**: Change LED size, spacing, and gap on-the-fly
- **Sound Design**: 8-bit style beeps and melodies generated with pygame
- **Smart AI**: Opponents that defend, pass, and shoot strategically

### Target Audience
Competition judges and gaming enthusiasts who appreciate:
- Retro gaming aesthetics
- Technical craftsmanship
- Interactive demonstrations
- Nostalgic 8-bit experiences

## Design Philosophy
- **Authentic LED Look**: Circular LEDs with glow effects and visible "off" states
- **Retro Gaming**: Pixel art, simple controls, classic game mechanics
- **Polish Over Features**: Better to have 1-2 excellent games than 4 mediocre ones
- **Interactive Demo**: Easy for judges to try both visual styles and game modes
